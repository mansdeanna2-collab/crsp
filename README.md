# 淘宝商城模板 (Taobao-style Shop Template)

一个仿淘宝风格的移动端商城页面模板，包含四个主要页面：首页、消息、购物车、我的。

## 📱 页面预览

### 首页 (Home)
- 顶部搜索栏（支持定位、搜索、扫码）
- 分类导航（10个商品分类）
- 轮播广告位
- 活动入口（限时秒杀、天天特价等）
- 商品推荐列表（瀑布流布局）

### 消息 (Messages)
- 消息分类Tab切换
- 消息列表展示
- 未读消息角标

### 购物车 (Shopping Cart)
- 按店铺分组的商品列表
- 商品规格展示
- 数量增减控制
- 单选/店铺全选/全选功能
- 实时价格计算
- 商品推荐
- 底部结算栏

### 我的 (Profile)
- 用户信息头部（头像、昵称、会员标识）
- 订单入口（待付款、待发货、待收货等）
- 我的资产（红包卡券、淘金币、余额、积分）
- 功能菜单列表

## 🚀 快速开始

### 直接打开
在浏览器中直接打开 `index.html` 文件即可预览。

### 本地服务器
```bash
# 使用Python启动简单服务器
python -m http.server 8080

# 或使用Node.js的http-server
npx http-server -p 8080
```

然后访问 `http://localhost:8080`

## 🎨 技术特点

- **纯前端实现**：HTML5 + CSS3 + JavaScript
- **响应式设计**：适配移动端和桌面端
- **无框架依赖**：原生JavaScript，无需构建工具
- **Font Awesome图标**：使用CDN加载图标库
- **渐变配色**：采用现代渐变色彩设计
- **交互动效**：按钮点击、页面切换等动画效果

## 📐 设计规范

- 主题色：`#ff5722` (橙红色)
- 字体：系统默认字体栈
- 最大宽度：480px（移动端优化）
- 底部导航高度：55px

## 🔧 自定义

### 修改主题色
在 `css/style.css` 中搜索 `#ff5722` 或 `#ff6b35` 替换为你想要的颜色。

### 添加商品
在 `index.html` 中的 `.product-grid` 区域添加新的商品卡片。

### 修改分类
在 `index.html` 中的 `.category-nav` 区域修改分类图标和名称。
